SHORT_DESCRIPTION

A modified version of the "Page Link" core field. Returns ID of post instead of page link.

EXTENDED_DESCRIPTION

A modified version of the "Page Link" core field. Returns ID of post instead of page link.

This ACF field type is compatible with:
* ACF 4

== Installation ==

1. Copy the `acf-post-id` folder into your `wp-content/plugins` folder
2. Activate the Post ID plugin via the plugins admin page
3. Create a new field via ACF and select the Post ID type
4. Please refer to the description for more info regarding the field type settings
